# Linux 磁盘与文件系统管理

系统管理员很重要的任务之一就是管理好自己的磁盘文件系统，每个分区槽不可太大也不能太小， 太大浪费，太小则不够用。

前面谈到的文件权限与属性，他们是记录在文件系统的哪个区块内？这就要谈到 filesystem 中的 inode 与 block 了。

为了虚拟化大容量磁盘，现在的 Centos7 默认使用大容量效能较佳的 xfs 当预设文件系统。

本章重点：如何制作文件系统，包括分区、格式化与挂载等。该章比较重要
